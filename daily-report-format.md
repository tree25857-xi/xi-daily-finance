## 專業財報格式結構

### 1. 封面標題區
```
┌─────────────────────────────────────────┐
│  🌸 晨曦金融網 Daily Report             │
│  日期：2026年4月9日                      │
│  市場：全球金融市場概況                   │
└─────────────────────────────────────────┘
```

### 2. 市場概覽區 (Market Overview)
```
┌─────────────────────────────────────────┐
│  📊 市場總覽                            │
│  ┌─────────┬─────────┬─────────┐       │
│  │ BTC     │ ETH     │ SOL     │       │
│  │ $71,147 │ $2,203  │ $83.27  │       │
│  │ +4.01%  │ +5.61%  │ +3.64%  │       │
│  └─────────┴─────────┴─────────┘       │
└─────────────────────────────────────────┘
```

### 3. 熱錢流向區 (Hot Money Flow)
```
┌─────────────────────────────────────────┐
│  🔥 熱錢溫度計                          │
│  • 美元指數 DXY    +0.3% (偏強)        │
│  • 恐慌指數 VIX    -1.2% (市場穩定)   │
│  • 比特幣現貨ETF   +3.0% (資金流入)    │
│  • 新興市場 EM     +$2.3B (淨流入)     │
└─────────────────────────────────────────┘
```

### 4. 區塊報價區 (Sector Quotes)
```
┌─────────────────────────────────────────┐
│  💰 加密貨幣         🇹🇼 台灣股市       │
│  ┌────┬────┬────┐   ┌────┬────┬────┐  │
│  │BTC │ETH │SOL │   │2330│TSM │TXF │  │
│  └────┴────┴────┘   └────┴────┴────┘  │
│                                         │
│  🤖 AI 科技股       📈 外匯            │
│  ┌────┬────┬────┐   ┌────┬────┬────┐  │
│  │NVDA│MSFT│AMD │   │EUR │JPY │GBP │  │
│  └────┴────┴────┘   └────┴────┴────┘  │
└─────────────────────────────────────────┘
```

### 5. 今日焦點新聞 (Top News)
```
┌─────────────────────────────────────────┐
│  📰 今日焦點                            │
│  1. 伊朗停火協議達成...                  │

│  3. 台積電 ADR 大漲 6.74%...            │
└─────────────────────────────────────────┘
```

### 6. 專業分析區 (Analysis)
